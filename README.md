## Personal Research

### [Covariance Matrix Estimation](https://github.com/TheoDom-272/CovMatrixEstimation)
**Covariance Matrix Estimation for Index Replication**

A Python framework for evaluating covariance matrix estimators applied to 
index replication via ex-ante tracking error minimization.

**Estimators compared :** Rolling Sample · EWMA · Ledoit-Wolf (LW) · ANLS · QIS

**Key findings :**
- QIS and LW consistently dominate Rolling and EWMA across all configurations
- The concentration ratio *c = p/n* is the principal source of estimation uncertainty
- Shrinkage estimators (QIS, LW) are substantially less sensitive to *c* than Rolling
- Daily data outperforms weekly for large-dimension equity universes

**Test universes :** MSCI EMU (~240 equities) · MSCI US (~550 equities)
